# IT Support & Ticketing API (FastAPI)

## 🚀 Overview
A simple REST API simulating IT Support operations such as user management, ticket handling, and system health monitoring.

## 🧰 Tech Stack
- Python
- FastAPI
- Logging

## ⚡ Features
- Create and retrieve users
- Create and track support tickets
- Health check endpoint

  ## 🔗 Real-World Context

This project reflects tasks performed in enterprise environments using tools like:
- Microsoft Intune
- Active Directory
- ServiceNow

It simulates how support engineers interact with backend systems and automate workflows.
